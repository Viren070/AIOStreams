title: Introduction

AIOStreams is a Stremio addon that aggregates streams from all of your other addons, then applies your filters, sorting, and formatting before passing results back to Stremio.

## Prerequisites

- A Stremio account and the Stremio app (desktop, Android TV, or web)
- A Debrid service (Real-Debrid, TorBox, AllDebrid, etc.) _or_ direct P2P / Usenet access
- Either a **[public instance](/getting-started/public-instances)**, a **[managed ElfHosted plan](https://store.elfhosted.com/product/aiostreams/elf/viren070/)**, or a machine to self-host on (VPS, NAS, or local PC)

## Quick start

<Steps>
  <Step>
    **Choose how to run AIOStreams.** The quickest option is using one of the
    available **[public instances](/getting-started/public-instances)** —
    no setup required. Alternatively, follow the
    [Deployment](/getting-started/deployment) guide to self-host with
    Docker.
  </Step>
  <Step>
    **Open** the configure page at `https://your-instance/stremio/configure` and
    walk through the setup wizard.
  </Step>
  <Step>**Install** the generated addon URL into Stremio.</Step>
</Steps>
